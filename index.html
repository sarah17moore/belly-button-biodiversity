Project Overview

This project collected demographic information of bacteria data from volunteers' belly button. Base on these data, we created a interactive dashboard to visulize each volunteer's belly button biodiversity.

By selecting Test Subject ID number from the dropdown menu, the individual's demographic information table and the following charts will update. The horizontal bar chart identifys the subject's top 10 bacterial species in their navel. The bubble chart displays bacteria culture test result for each sample. The gauge chart reveals reveals how often does the subject clean his belly button.
